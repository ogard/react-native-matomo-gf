# react-native-matomo-gf

React Native wrapper for Matomo iOS and Android SDK

## Installation

```sh
npm install react-native-matomo-gf
```

## Usage

```js
import { multiply } from 'react-native-matomo-gf';

// ...

const result = await multiply(3, 7);
```
